# Getting Started

### Setup
* Ubuntu 20.xx
* OpenJDK 11.xx
* set **JAVA_HOME** env var
* Apache Maven 3.6.x
* set **M2_HOME** env var
* install curl

### How to run
```mvn clean install```


```mvn spring-boot:run```

### Testing
```curl localhost:5151```

Answer **Ok** or **Not Ok** <br />

Feel free to change URL in application.properties
